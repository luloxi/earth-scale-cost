Methodology and reliability

The page no longer depends on a blank first load. It ships with an initial embedded ranking snapshot, then updates clientside from live sources.

Embedded snapshot data ensures the landing page always shows the ranking immediately.
A parallel clientside refresh updates tracked chains every 30 seconds and reranks automatically.
Most rows use Chainspect chain pages. XRP uses Blockchair. Unichain uses Blockscout.
Observed live TPS is not the same as guaranteed sustainable TPS. This is a live market-style ranking, not a full protocol stress test.
